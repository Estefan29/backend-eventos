# Manual de Usuario -Eventos USC

### Registro y Login

Para crear tu ceunta:

1. Ingresa tu nombre, correo y contraseña 
opcinalmente, puedes indicar tu rol ( por defecto sera estudiante)
2. el nombre debe tener minimo 3 caracteres.
- El correo debe ser real y valido 
- La contraseña debe tener por lo menos 6 caracteres 
3. Envia el formulario de registro

No pueder volver a registrarte con un correo que ya existe.

### Iniciar sesión

Para acceder a tu cuenta:

1. Ingresa tu correo y contraseña con los que te registraste
2. Envia el formulario de login
3. si los datos son correctos, podras ingresar a la plataforma 
 
 ### Ver tu perfil

 Una vez que hayas iniciado sesión, puedes ver tu información personal:

- Nombre completo
- Correo electronico
- Telefono
- Rol
- Carrera o programa
- Rol actual

### Actualizar tu perfil

Puedes actualizar tu información personal:

1. Modifica los campos que necesitas:
- Nombre 
- Telefono
- Carrera
- Foto de perfil
- Correo electronico
2. Envia los cambios

presiona **Guardar cambios** y podras ver tu perfil actualizado

### Recuperar contraseña 
 
Si olvidaste tu contraseña:

1. Ingresa tu correo en el formulario de recuperación
2. Recibirás un correo con un enlace para restablecer tu contraseña.
3. Haz clic en el enlace y crea una nueva contraseña.
4. Verás el mensaje: 

El enlace de recuperación tiene **vigencia de 1 hora**. Si expira, solicita un nuevo enlace.
