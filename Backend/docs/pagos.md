# Gestión de Pagos
Bienvenido a la sección de pagos. Aqui aprenderas como gestionar tus pagos de forma sencilla.
### Ver todos tus pagos 
1. En esta sección puedes consultar todos los pagos que has realizado o que estan pendientes 
- Qué Veras:
Una lista con cada pago, mostrando:
-Monto
-Estado (Pendiente, Completado o cancelado)
- Fecha del pago 
- Metodo de pago
### Crear u nuevo pago
Cuando te inscribes a un evento o servicio que requiere pago
- Que veras:
Un formulario para registar tu pago, donde debes indicar:
- Monto
- Método de pago (Tarjeta, efectivo, etc.)
