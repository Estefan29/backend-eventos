# Gestión de Usuarios

En esta sección aprenderás cómo funciona la gestión de usuarios dentro de la plataforma. Esto incluye registrar nuevos usuarios, ver los existentes, actualizar su información y eliminarlos.

### 1. Registrar un nuevo usuario


Cuando quieras agregar un nuevo usuario al sistema, este es el proceso.

**Cómo funciona:**
- Proporciona los datos del usuario (nombre, correo, teléfono, carrera, rol,)  .
- Envías la información. 
- El usuario queda registrado y listo para usar la plataforma.

### 2. Ver todos los usuarios

Puedes consultar la lista de usuarios registrados en la plataforma.

**Cómo funciona:**
- Puedes aplicar filtros por rol o realizar una búsqueda por nombre o correo.
- La lista se muestra paginada para facilitar la visualización.


### 4. Actualizar información de un usuario

Si necesitas modificar datos de un usuario, este es el flujo.

**Cómo funciona:**

- Puedes actualizar nombre, teléfono, carrera, rol o estado activo.
- El sistema aplica los cambios en la base de datos y confirma la actualización.


### 5. Eliminar un usuario
**Cómo funciona:**

- Buscas el usuario por su ID.
- El sistema elimina el registro de la base de datos.
- Solo se puede eliminar si el usuario existe; si no, recibirás un mensaje de error.
